# JavaScript Core Concepts

<h3>Essential Programming Concepts</h3><p><strong>Variables:</strong> Store data using let, const, or var. Modern JavaScript prefers const for constants and let for variables.</p><p><strong>Functions:</strong> Write reusable code blocks with function declarations or arrow syntax. Arrow functions provide cleaner syntax for short operations.</p><p><strong>Arrays:</strong> Store multiple values in one variable. Use methods like map, filter, and reduce for powerful data transformations.</p><p><strong>Objects:</strong> Represent real-world entities with key-value pairs. Objects are fundamental to JavaScript programming.</p><p><strong>DOM Manipulation:</strong> Interact with web page elements dynamically using methods like querySelector and addEventListener.</p>

# Instructions for Users

<h2>How to Format Your Notes</h2><p>This notes app supports rich HTML formatting to help you create well-structured and visually appealing notes. Here are the recommended HTML tags to use:</p><h3>Text Formatting</h3><ul><li>Use p tags for paragraphs</li><li>Use strong or b tags for bold text</li><li>Use em or i tags for italic text</li><li>Use code tags for inline code snippets</li></ul><h3>Headings</h3><ul><li>Use h2 for main section headings</li><li>Use h3 for subsection headings</li><li>Use h4 for smaller subheadings</li></ul><h3>Lists</h3><ul><li>Use ul and li tags for unordered (bullet) lists</li><li>Use ol and li tags for ordered (numbered) lists</li></ul><h3>Best Practices</h3><ul><li>Keep titles short and descriptive</li><li>Use tags to categorize notes for easy searching</li><li>Structure content with headings for better readability</li><li>Separate multiple notes with --- divider</li></ul><p><em>Note:</em> The app automatically handles creation and update timestamps, so you don't need to include them when importing notes.</p>
